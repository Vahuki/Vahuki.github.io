Mô tả:

  1. Tạo mới một file có tên index.html.
  2. Thêm một hình ảnh vào bên trong thẻ <body>; (học viên có thể chọn bất kỳ hình ảnh yêu thích).
  3. Nếu không có hình ảnh, tải về hình ảnh theo link:Tải hình ảnh.
  4. Sử dụng thuộc tính alt để mô tả hình ảnh.
